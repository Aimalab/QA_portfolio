# QA Portfolio — saucedemo.com

Автотесты для демонстрации навыков QA-инженера.  
Тестируемый сайт: https://www.saucedemo.com

## Стек
- Python 3.13
- Selenium
- Pytest
- WebDriver Manager

## Структура проекта

```text
qa-portfolio/
├── tests/
│   ├── test_login.py        # тесты авторизации
│   └── test_purchase.py     # тесты покупки
├── pages/
│   ├── login_page.py        # Page Object: страница входа
│   ├── inventory_page.py    # Page Object: каталог товаров
│   ├── cart_page.py         # Page Object: корзина
│   └── checkout_page.py     # Page Object: оформление заказа
├── docs/
│   └── bug_reports.md       # баг-репорты
├── conftest.py              # конфигурация браузера
└── README.md
```

## Запуск тестов

# установить зависимости
pip install selenium pytest webdriver-manager

# запустить все тесты
pytest tests/ -v

# запустить только авторизацию
pytest tests/test_login.py -v

# запустить только покупку
pytest tests/test_purchase.py -v

## Паттерны и подходы
- Page Object Model (POM) — логика страниц отделена от тестов
- Явные ожидания (WebDriverWait) — стабильность на медленных страницах
- Фикстуры Pytest — автоматический запуск и закрытие браузера
- Негативное тестирование — проверка граничных случаев и ошибок
