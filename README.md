# QA Portfolio — saucedemo.com

Тестовый проект для демонстрации навыков QA-инженера.

## Стек
- Python 3.12
- Selenium
- Pytest
- WebDriver Manager

## Структура проекта
qa-portfolio/

├── tests/          # автотесты

├── pages/          # Page Object Model

├── docs/           # баг-репорты и документация

└── conftest.py     # конфигурация браузера

## Что тестируем
Сайт: https://www.saucedemo.com  
Модуль: Форма авторизации

| ID | Сценарий | Тип |
|----|----------|-----|
| TC-001 | Успешная авторизация | Позитивный |
| TC-002 | Неверный пароль | Негативный |
| TC-003 | Пустые поля | Негативный |
| TC-004 | Заблокированный пользователь | Негативный |
| TC-005 | Только логин без пароля | Негативный |

## Запуск тестов

```bash
pip install -r requirements.txt
pytest tests/ -v
```
