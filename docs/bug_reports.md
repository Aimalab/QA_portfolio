# Bug Reports — saucedemo.com

## BUG-001: Нет маскировки пароля при автозаполнении

**Статус:** Open  
**Приоритет:** Средний  
**Компонент:** Форма входа  
**Окружение:** Chrome 124, Windows 11  
**Воспроизводимость:** Всегда  

### Шаги для воспроизведения
1. Открыть https://www.saucedemo.com
2. Разрешить браузеру автозаполнить логин и пароль
3. Наблюдать за полем пароля

### Ожидаемый результат
Пароль отображается скрытым (точки), даже при автозаполнении

### Фактический результат
На долю секунды пароль отображается открытым текстом до маскировки

### Связанный тест
TC-002

## BUG-002: Сообщение об ошибке не исчезает после исправления логина

**Статус:** Open  
**Приоритет:** Низкий  
**Компонент:** Форма входа  
**Окружение:** Chrome 124, Windows 11  
**Воспроизводимость:** Всегда  

### Шаги для воспроизведения
1. Открыть https://www.saucedemo.com
2. Ввести неверный логин и пароль → появляется ошибка
3. Начать исправлять логин в поле

### Ожидаемый результат
Сообщение об ошибке скрывается когда пользователь начинает вводить текст

### Фактический результат
Сообщение об ошибке остаётся на экране пока не нажать крестик вручную

### Связанный тест
TC-002, TC-003
